# Mega-menu (in header)

<p class="lead">Global navigation links to different pages within the same "entrance"</p>

## When to use

- In pages that are part of either [kth.se](https://kth.se), [intranet](https://intra.kth.se) or [student web](https://kth.se/student).

## When not to use

- For pages that are not part of any entrance.
- Do not use this component as a local menu
